# C - Sorting Algorithms & Big O

## Description

This project focuses on understanding and implementing several sorting algorithms in C, as well as analyzing their time complexity using Big O notation.

Sorting algorithms are fundamental in computer science because they allow efficient organization of data. Understanding how they work helps developers choose the best algorithm depending on the size and nature of the data.

The main goal of this project is to:

- Implement different sorting algorithms
- Understand how each algorithm works internally
- Learn how to analyze algorithm performance using Big O notation
- Understand differences between data structures such as arrays and linked lists
- Identify stable and unstable sorting algorithms

---

## Learning Objectives

At the end of this project, you should be able to explain:

- At least four different sorting algorithms
- What is Big O notation
- How to evaluate time complexity
- The difference between best case, average case and worst case complexity
- What is a stable sorting algorithm
- When to choose one sorting algorithm over another
- The difference between arrays and linked lists when sorting
